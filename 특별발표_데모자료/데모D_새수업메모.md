보고서 검토수정 차시 준비 메모 (강좌 마지막에 넣을 차시)

- 신입들이 초안 쓰자마자 그냥 제출해버림. 검토를 거의 안 함 ㅠㅠ
- 핵심 포인트 = 검토는 "다른 사람 눈으로" 내 글을 다시 읽는 것
- 셀프 체크리스트 하나 만들어주면 좋을듯 - 결론이 맨 앞에 있나 / 근거가 결론을 받쳐주나 / 오타 없나 / 숫자에 출처 있나
- 시간 - 강의 20분, 짝꿍 피어리뷰 실습 30분, 공유 10분
- 실습은 짝꿍이랑 보고서 서로 봐주기. 포스트잇으로 코멘트 달기
- 준비물 = 검토 체크리스트 인쇄물, 포스트잇, 빨간펜
- 학습목표 - 체크리스트로 자기 보고서 점검할수있다 / 동료 피드백 주고받을수있다 / 받은 피드백 반영해서 고칠수있다
- 피드백 받을때 방어하지 말고 일단 듣기. 이것도 한마디 해주기
- 좋은 피드백 vs 나쁜 피드백 예시 보여주기
- 이게 강좌 마지막 차시임. 다음 예고 없음
- 과제 = 자기 보고서 피어리뷰 받아서 고친 최종본 제출하기
