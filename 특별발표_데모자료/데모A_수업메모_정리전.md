신입 보고서교육 1차시 준비 메모

- 일단 아이스브레이킹부터... 자기가 최근에 쓴 보고서 하나 떠올려보라고 하기. 한 5분?
- 핵심은 "보고서는 글짓기가 아니다" 이거 꼭 강조하기. 목적이 있는 문서라는 거.
- 두괄식!!! 결론 먼저!! 신입들이 제일 약한 부분임
- 보고서 기본 4덩어리 = 목적 / 결론 / 근거 / 실행(액션). 이건 그림으로 보여주면 좋을듯
- 시간배분 대충 - 강의 25분, 조별실습 20분, 공유랑 질문 10분 정도
- 준비물 : 예시 보고서 2개(좋은거 하나 나쁜거 하나), 구조잡기 워크시트, 포스트잇
- 실습은 나쁜 보고서를 4요소 구조로 고쳐쓰기. 조별로 하기
- 아 그리고 보고서 쓰기 전체 흐름도 한번 짚어주자 - 자료수집 → 구조잡기 → 초안 → 검토수정 → 제출. 오늘은 '구조잡기'가 메인
- 학습목표 세개정도 - 4가지 구성요소 안다 / 두괄식으로 뼈대 잡을수있다 / 목적에 맞는 구조 고른다
- 참고링크 https://example.com/report-tips (볼지 말지 고민)
- 다음차시 예고 = 문장쓰기
- 나쁜 예시는 작년에 돌던 그 보고서 익명처리해서 쓰면 될듯
- 과제 내주기 - 자기가 쓴 보고서 하나 4요소 구조로 다시 정리해오기
- 질문받는 시간도 좀 남기기
